# 🏭 Automatización con PLC S7-1200 y Factory I/O

## 📌 Descripción

Implementación de un sistema de control utilizando un **PLC S7-1200**, una **HMI** y simulación en **Factory I/O**, con comunicación mediante **OPC Server** y conexión **Ethernet punto a punto**.

---

## 🎯 Objetivo

Diseñar y validar un sistema de control en simulacion, integrando lógica PLC, visualización HMI y comunicación en red REAL.

---

## ⚙️ Tecnologías utilizadas

* PLC Siemens S7-1200
* HMI (interfaz hombre-máquina)
* Factory I/O
* OPC Server (comunicación de datos)
* Ethernet punto a punto

---

## 🔌 Funcionamiento

* El PLC fisico ejecuta la lógica de control del sistema
* Factory I/O simula el entorno industrial
* La HMI fisico permite supervisión e interacción
* Comunicación en tiempo real mediante OPC

---

## 💡 Resultado

Sistema funcional de automatización con integración completa entre simulación, control y monitoreo.
